## 📂 Project Structure

### 🟦 [1. Manual Testing](theedanaisronprempree-tester/My-Tester-Portfolio/1_Manual_Testing)
เน้นทักษะการคิดวิเคราะห์และการออกแบบกรณีทดสอบให้ครอบคลุม
* **Highlight:** การเขียน Test Case สำหรับระบบ Register, Test Scenario การจองเรือ และการทำ Defect Report

### 🟪 [2. SQL](theedanaisronprempree-tester/My-Tester-Portfolio/2_SQL)
เน้นทักษะการตรวจสอบข้อมูลหลังบ้าน (Backend Validation)
* **Highlight:** การเขียนคำสั่ง Select, Join เพื่อดึงข้อมูลมาเปรียบเทียบกับหน้าจอ UI และการจัดการฐานข้อมูลเบื้องต้น

### 🟧 [3. Automated Test (Web UI)](theedanaisronprempree-tester/My-Tester-Portfolio/3_Automated_Test(Web_UI))
เน้นการเพิ่มประสิทธิภาพในการทดสอบซ้ำ ๆ (Regression Testing)
* **Highlight:** ใช้ **Robot Framework** ในการ Automate การค้นหางานบน JobThai พร้อมการจัดการ Log และ Screenshots

### 🟩 [4. API Testing](theedanaisronprempree-tester/My-Tester-Portfolio/4.1_API_Testing_Postman_Basic_User-Petstore.pdf)
เน้นการทดสอบประสิทธิภาพและความถูกต้องของข้อมูลระหว่างระบบ
* **Highlight:** ใช้ **Postman** ร่วมกับ **Swagger Petstore**, การสุ่มข้อมูล (Dynamic Data) และการทำ Schema Validation
